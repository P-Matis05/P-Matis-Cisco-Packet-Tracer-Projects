# P Matis - Cisco Packet Tracer Projects

Collection of Cisco Packet Tracer networking labs and enterprise network simulations completed through Cisco Networking Academy learning.

This repository focuses on practical networking, infrastructure, troubleshooting, and network security concepts relevant to enterprise environments and cyber security pathways such as SOC, Blue Team, and infrastructure security.

---

# Skills Demonstrated

## Networking Fundamentals
- Network topology design
- Logical and physical network representation
- TCP/IP and OSI models
- Connectivity testing
- Interface configuration

## Routing & Switching
- VLAN implementation
- Inter-VLAN routing
- Router-on-a-Stick configuration
- OSPFv2 routing
- Switch configuration
- Router configuration

## IP Addressing
- IPv4 subnetting
- VLSM addressing schemes
- Enterprise IP allocation

## Network Security
- Standard IPv4 ACLs
- Traffic filtering
- Network segmentation
- Basic security controls

## Infrastructure Services
- DHCP configuration
- Static NAT configuration
- Wireless networking
- WAN serial connections

## Troubleshooting
- Connectivity diagnostics
- End-to-end communication testing
- Enterprise troubleshooting scenarios
- Integrated networking challenges
